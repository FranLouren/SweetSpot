## 📌 Descripción
Aplicación web para la reserva de pistas de pádel online.  
Permite a los usuarios registrarse, iniciar sesión, reservar pistas en fechas y horas específicas, 
evitar solapamientos, y gestionar sus reservas.

---

## 🛠️ Tecnologías usadas

- HTML, CSS, JavaScript
- Backend: PHP 
- Base de datos: MySQL
- Control de versiones: Git + GitHub

---

## 📁 Estructura del proyecto

/frontend/ → Interfaz de usuario
/backend/ → Lógica del servidor y API
/database/ → Scripts y estructura de la base de datos
/docs/ → Documentación y memoria del TFG


---

## 🚀 Cómo ejecutar el proyecto

1. Clona el repositorio
2. Instala XAMPP o Node.js (según backend)
3. Importa `estructura.sql` en tu base de datos local
4. Lanza el servidor
5. Abre `frontend/index.html` en el navegador
